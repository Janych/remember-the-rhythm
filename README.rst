====================
Remember The Rhythm
====================

A plugin for rhythbox to remember last playing song and playback time.

-------------
Requirements
-------------

Rhythmbox 2.90 or above

-------------
Installation
-------------


Ubuntu Repository
~~~~~~~~~~~~~~~~~~

::

    sudo add-apt-repository ppa:loneowais/ppa
    sudo apt-get update
    sudo apt-get install remember-the-rhythm


From Source
~~~~~~~~~~~~

Install:
````````

::

    sudo make install

Remove
```````

::

     sudo make uninstall
